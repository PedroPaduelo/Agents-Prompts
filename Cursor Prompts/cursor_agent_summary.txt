O arquivo `cursor agent.txt` descreve um assistente de codificação de IA agentic, "Claude 3.7 Sonnet", que opera no IDE Cursor. Ele atua como um programador de pares para auxiliar o usuário em tarefas de codificação, como criar, modificar, depurar código ou responder a perguntas. O documento detalha regras para o uso de ferramentas pela IA (como chamá-las, não se referir aos nomes das ferramentas para o usuário, e chamar apenas quando necessário), para fazer alterações no código (usar ferramentas de edição, não gerar código diretamente para o usuário, garantir que o código seja executável, criar arquivos de gerenciamento de dependências e READMEs, preferir edições menores e ler o arquivo antes de editar), e para pesquisa e leitura (preferir busca semântica, ler grandes seções de arquivos e evitar loops de correção de erros). Ele também lista as funções das ferramentas disponíveis para a IA, incluindo busca de código, leitura de arquivo, execução de comandos de terminal, listagem de diretórios, busca grep, edição de arquivo, busca de arquivo, exclusão de arquivo, reaplicação de edições e busca na web.
